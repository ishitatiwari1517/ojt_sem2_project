# EnergyLens – OJT Project

EnergyLens is a web-based system that provides insights into household electricity consumption.

## Features (Planned)
- Appliance-level usage tracking
- Cost estimation
- Insights and alerts

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

- Repository initialized
- Backend scaffolded
- Frontend scaffolded
- Database connection setup
- Health API implemented
